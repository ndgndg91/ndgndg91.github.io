Base64 Encode/Decode

Base64는 텍스트나 파일을 64개의 문자로 변환하여 데이터를 안전하게 전송하거나 저장할 수 있게 하는 방법입니다.



URL Encode/Decode



URL 인코딩은 URL에서 특정 문자를 퍼센트 기호(%)와 두 자리 16진수로 바꾸는 것을 의미합니다.

UUID 생성기

UUID(Universally Unique Identifier)는 고유한 식별자를 생성하는 데 사용됩니다. 각 버전은 서로 다른 생성 방식과 용도를 가지고 있습니다.

UUID v4 (랜덤 기반)

완전히 랜덤으로 생성되며, 가장 널리 사용됩니다. 충돌 가능성이 매우 낮습니다.

결과: 00000000-0000-0000-0000-000000000000

UUID v7 (시간순 정렬 가능)

유닉스 타임스탬프와 랜덤 값을 결합하여 생성됩니다. 시간순 정렬이 가능하며 데이터베이스 인덱싱에 유리합니다.

결과: 00000000-0000-0000-0000-000000000000

UUID v1 (시간 기반)

현재 시간과 MAC 주소를 기반으로 생성됩니다. 고유하지만 시간 정보를 노출할 수 있습니다.

결과: 00000000-0000-0000-0000-000000000000

UUID v5 (이름 기반, SHA-1)

네임스페이스와 이름 문자열을 SHA-1 해시로 변환하여 생성합니다. v3보다 보안성이 높습니다.

결과: 00000000-0000-0000-0000-000000000000

UUID v3 (이름 기반, MD5)

네임스페이스와 이름 문자열을 MD5 해시로 변환하여 생성합니다. 동일한 입력에 대해 항상 동일한 결과가 나옵니다.

결과: 00000000-0000-0000-0000-000000000000

Random Hex 생성기

랜덤한 16진수 문자열을 생성합니다. 길이를 입력하여 원하는 크기의 hex 문자열을 얻을 수 있습니다.

결과: 00000000

Unix Timestamp

Unix Timestamp는 1970년 1월 1일 00:00:00 UTC(협정 세계시)부터 경과한 초 단위의 시간을 나타냅니다. 이를 "에포크(Epoch)"라고도 부르며, 주로 컴퓨터 시스템에서 시간을 기록하거나 계산할 때 사용됩니다.

현재 Unix Timestamp: 0
현재 타임존: Loading...
선택한 타임존 시간: YYYY MM DD HH MM SS

AES Encrypt/Decrypt

AES(Advanced Encryption Standard)는 대칭 키 암호화 알고리즘으로, 동일한 키를 사용하여 데이터를 암호화하고 복호화합니다.





RSA Encrypt/Decrypt

RSA는 공개 키와 개인 키를 사용하는 비대칭 암호화 알고리즘입니다. 공개 키로 암호화하고 개인 키로 복호화합니다.
